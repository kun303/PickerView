# PickerView-master
One about like IOS wheelView packaging Library

#How to
>Step 1. Add the JitPack repository to your build file
```java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

>Step 2. Add the dependency
```java
dependencies {
	        implementation 'com.github.newsolf:PickerView:1.0.0'
	}
```
